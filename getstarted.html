What is Inside?

This website will contain simple tutorials for a variety of things within Cricket. It won't go in depth all the way because some of these skills are ones that I don't even know, such as Leg-Spin, Inswingers, or Outswingers. Also, fielding will only contain tips for catching practice or maybe a few running exercises. Batting and bowling will be using videos as well to make it easier to tell what to do. All of these lessons are of what I think is needed to do, so it might be ocassionally incorrect.
